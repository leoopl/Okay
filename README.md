## Run Locally

Clone o projeto

```bash
  git clone https://github.com/leoopl/Okay.git
```

Vá até a pasta do projeto

```bash
  cd Okay
```

Instale as dependencias

```bash
  yarn install
```

Execute o projeto

```bash
  yarn dev
```

Abra o Projeto

```bash
  code .
```
